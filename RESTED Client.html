You can include dynamic elements into your URLs. Simply type{{something}} into your URL, and then add something as a key here, along with a value for that key, and it will be resolved for you. This value will be resolved across all requests you have saved, making it great for things like API keys or IDs that you may have to change in several URLs at once.

KeyValue


              {
                example: "json",
                soThatYouCan: 7357,
                your: "HIGHLIGHT_STYLE"
              }


Here you can import data from other sources. Two formats are currently supported, HTTP Archive (HAR) and Postman json.

HAR is a format that can be exported directly from your favourite browser's network tools. This means you can do a request you would like to recreate or alter at a later date, and import it to your collected requests. See this link for details.

An option to import data from Postman (Chrome extension) is also provided. In order to fetch a collection from postman, click the three dots next to a postman collection, click download, and copy the file's contents into the field below.


Here you can export data to other sources. Two formats are currently supported, HTTP Archive (HAR) and Postman json.

HAR is a format that can be exported directly from your favourite browser's network tools. Click here to learn more.

Postman is a popular Google Chrome extension that is a REST client, just like RESTED. By using this export, you can cooperate with colleagues that use Postman, dispite them working with different tools.
